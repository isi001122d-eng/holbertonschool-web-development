# CSS Advanced - Corporate Landing Page Design

This section covers the advanced CSS styling implementation for our corporate educational landing page based on the professional Figma design specification.

## Core Visual Requirements Implemented:
- **Navigation:** Transparent navigation bar positioned at the top of the page with the logo aligned left and 3 white bold navigation links (COURSES, PRICING, LOGIN) aligned right.
- **Hero Section:** Dark background image enhanced with a precise radial gradient overlay, centering the large bold heading "Get schooled".
- **Keywords Matrice:** Horizontal row displaying three focus keywords: SMILES, GRIN, and LAUGH.
- **Call To Action:** Centered purple rounded "REGISTER FOR FREE" dynamic button.
- **Sub-features & Team:** "Learn from the pros" section with custom purple typography accents alongside 4 circular profile images aligned horizontally, complete with bold names and italic descriptions.
